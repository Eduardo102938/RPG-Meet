# 🐉 RPG Meet

Aplicativo Android para sessões de RPG online com videoconferência integrada via Jitsi Meet.

---

## 🚀 Como rodar

1. Abra no Android Studio
2. Sincronize o Gradle
3. Execute no emulador ou celular

---

## 📂 Estrutura

- MainActivity → Entrada da sala
- CallActivity → Videoconferência
- DiceManager → Rolagem de dados

---

## 🎯 Funcionalidades

- Videoconferência (Jitsi)
- Rolagem de dados (D20)
- Entrada por nome de sala

---

## 🧠 Prompt usado

Criação de app Android RPG com videoconferência usando Jitsi Meet.
